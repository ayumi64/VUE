# my_vue

> vue learning

## Build Setup

# npm较慢 使用cnpm代理
$ npm install -g cnpm --registry=https://registry.npm.taobao.org
$ npm config set registry https://registry.npm.taobao.org

# 全局安装 vue-cli
$ cnpm install --global vue-cli
# 创建一个基于 webpack 模板的新项目
$ vue init webpack my-project

## 启动
$ cd my-project
$ cnpm install
$ cnpm run dev
